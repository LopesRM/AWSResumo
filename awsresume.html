Cloud Computing

Cloud computing é a entrega de serviços de computação pela internet, incluindo armazenamento, processamento e redes, sem a necessidade de gerenciar infraestrutura física.

O que é AWS

A AWS (Amazon Web Services) é uma plataforma de serviços de computação em nuvem que oferece soluções escaláveis e flexíveis para empresas e desenvolvedores.

Preços da AWS

A AWS oferece um modelo de preços baseado no uso, permitindo que você pague apenas pelos serviços que utiliza, sem custos iniciais ou compromissos de longo prazo.

Visão Geral da Infraestrutura AWS

A infraestrutura global da AWS é composta por regiões, zonas de disponibilidade e pontos de presença para garantir alta disponibilidade e baixa latência.

Modelo de Responsabilidade Compartilhada

O modelo de responsabilidade compartilhada da AWS define as responsabilidades do cliente e da AWS em relação à segurança e conformidade.

Confiabilidade e Alta Disponibilidade

A AWS oferece serviços projetados para garantir alta disponibilidade e confiabilidade, como balanceamento de carga e replicação de dados.

Transição de um Data Center para a Nuvem

A transição de um data center para a nuvem envolve a migração de aplicativos e dados para aproveitar os benefícios da escalabilidade e flexibilidade da AWS.

APIs e REST

APIs REST são interfaces que permitem a comunicação entre sistemas usando métodos HTTP, amplamente utilizadas para integrar serviços na AWS.

Hospedar um Site Estático no Amazon S3

O Amazon S3 permite hospedar sites estáticos de forma simples e escalável, com suporte para HTML, CSS e JavaScript.

Visão Geral do Armazenamento na Nuvem

O armazenamento na nuvem oferece soluções flexíveis e escaláveis para armazenar e acessar dados de qualquer lugar.

Amazon Elastic Block Store

O Amazon EBS fornece volumes de armazenamento em bloco persistentes para uso com instâncias do Amazon EC2.

Armazenamento de Instância do EC2

O armazenamento de instância do EC2 oferece armazenamento temporário diretamente conectado à instância, ideal para dados transitórios.

Amazon Elastic File System

O Amazon EFS é um sistema de arquivos escalável e elástico para uso com instâncias do Amazon EC2.

Amazon S3 Glacier

O Amazon S3 Glacier é um serviço de armazenamento de baixo custo para arquivamento de dados e backup de longo prazo.

AWS Storage Gateway

O AWS Storage Gateway conecta ambientes locais à nuvem, permitindo armazenamento híbrido e backup eficiente.

Amazon VPC

O Amazon VPC permite criar redes virtuais isoladas logicamente na nuvem AWS, garantindo segurança e controle.

Security Groups

Os Security Groups atuam como firewalls virtuais para controlar o tráfego de entrada e saída de instâncias do EC2.

Amazon Route 53

O Amazon Route 53 é um serviço de DNS escalável e altamente disponível para gerenciar nomes de domínio.

Amazon CloudFront

O Amazon CloudFront é uma rede de distribuição de conteúdo (CDN) que entrega dados com baixa latência e alta velocidade.

Amazon EC2

O Amazon EC2 fornece capacidade de computação escalável na nuvem, permitindo criar e gerenciar instâncias de máquinas virtuais.

AWS Lambda

O AWS Lambda permite executar código sem a necessidade de gerenciar servidores, ideal para aplicações serverless.

AWS IAM

O AWS Identity and Access Management (IAM) permite gerenciar o acesso seguro aos recursos da AWS.

AWS CloudFormation

O AWS CloudFormation permite criar e gerenciar recursos da AWS usando modelos de infraestrutura como código.

Amazon CloudWatch

O Amazon CloudWatch monitora recursos da AWS e aplicações, fornecendo métricas e logs em tempo real.

AWS Organizations

O AWS Organizations permite gerenciar várias contas da AWS de forma centralizada, com políticas e controle de custos.

AWS Well-Architected Framework

O AWS Well-Architected Framework fornece boas práticas para projetar e operar sistemas confiáveis, seguros e eficientes na nuvem.

AWS Command Line Interface

A AWS CLI permite gerenciar serviços da AWS diretamente do terminal, automatizando tarefas administrativas.

AWS Systems Manager

O AWS Systems Manager fornece uma interface unificada para gerenciar recursos da AWS e ambientes locais.

AWS Step Functions

O AWS Step Functions permite orquestrar fluxos de trabalho distribuídos usando estados e transições.

AWS CloudTrail

O AWS CloudTrail registra atividades de API e eventos de gerenciamento para auditoria e conformidade.

AWS Config

O AWS Config monitora e registra alterações na configuração de recursos da AWS para auditoria e conformidade.

AWS Trusted Advisor

O AWS Trusted Advisor fornece recomendações para otimizar custos, desempenho e segurança na AWS.

Amazon Athena

O Amazon Athena permite consultar dados diretamente no Amazon S3 usando SQL, sem necessidade de infraestrutura.
