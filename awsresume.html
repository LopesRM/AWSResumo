Bem-vindo à AWS Adventure

Explore os serviços da AWS e aprenda de forma interativa e divertida!

EC2 (Elastic Compute Cloud)

O Amazon EC2 fornece capacidade de computação escalável na nuvem. Ele permite criar e gerenciar instâncias de máquinas virtuais para executar suas aplicações.

S3 (Simple Storage Service)

O Amazon S3 é um serviço de armazenamento de objetos seguro e escalável. Ideal para backups, arquivamento e análise de big data.

RDS (Relational Database Service)

O Amazon RDS é um serviço gerenciado para bancos de dados relacionais como MySQL, PostgreSQL e SQL Server.

AWS Lambda

O AWS Lambda permite executar código sem a necessidade de gerenciar servidores. Ideal para aplicações serverless.

CloudFront

O Amazon CloudFront é uma rede de distribuição de conteúdo (CDN) que entrega dados com baixa latência e alta velocidade.
